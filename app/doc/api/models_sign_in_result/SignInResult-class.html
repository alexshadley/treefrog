SignInResult enum

Possible results of a user sign-in.

Constants

FAILURE → const SignInResult

The sign-in attempt failed for an unknown reason.

const SignInResult(4)
INCORRECT_METHOD → const SignInResult

The user attempted to sign in using the wrong method. For example, if the account was created through a Google account, the user cannot sign in later using the same email through Facebook.

const SignInResult(1)
INCORRECT_PASSWORD → const SignInResult

The user entered an incorrect password. This only applies for Email sign-ins.

const SignInResult(3)
NONEXISTENT_USER → const SignInResult

The user doesn't exist in the database.

const SignInResult(2)
SUCCESS → const SignInResult

The user is allowed to sign in. This indicates that:

  • The user exists
  • The user used the correct method in signing in
  • The user entered the correct password, if applicable
