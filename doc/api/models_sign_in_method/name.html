name function

String name (SignInMethod m)

Gets a string representation of the name of a sign-in method. For example, for name(SignInMethod.Google) == "Google".

Implementation 

String name(SignInMethod m) {
  var s = m.toString();
  return s.substring(s.indexOf('.') + 1);
}
